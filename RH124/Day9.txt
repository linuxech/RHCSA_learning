Special Permissions
-----------------------------------------------------------------
Umask 
umask (user file-creation mode mask) is a command and a system setting that determines the default file permissions 
for newly created files and directories. 

	777(Full Permission)
-	022  -- Gives the permission as default  with octal value 755	
-	002  -- Gives the permission as default with octal value 775

~/.bashrc -- You can modify the file to change the umask value.
