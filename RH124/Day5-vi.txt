$Modes in vim$

Command Mode-- Initial Mode.
Extended Mode-- : -- help to access the Extended Mode. 
Edit Mode-- Insert Mode -- I 
Visual Mode -- Visual Edit Mode
		- Character Mode- Use the character selection in Visual Mode
		- Line Mode - Select the entire line.
		- Block Mode- Select the block of character.

Command Mode Option:

yy -- Yanked/Copy
dd -- Cut/Delete
p -- Paste
u -- Undo
Ctrl + r -- Redo
i -- Insert
o -- To insert after an empty line
gg -- Move cursor on Top of the file.
G -- Move custor to the bottom of the file. 

For more Operation using Extended Command mode

:18 -- To move the cursor to the line 18.
dn -- n represent the number of line you want to delete. Start counting from 0.
yn -- n represent the number of line you want to copy. Start counting from 0.
:q -- To quit file without saving
:w -- To write the changes to the file.
:q! -- quit file forcefuly. 

:set number -- Display the number of lines
