SELinux	Port Labeling

Service			Port Number
http			80
https			443
ssh			22


For Example: You need to change the port for apache from 80 to 82.

systemctl status httpd		\\Check the assigned port number.
vim /etc/httpd/conf/httpd.conf		\\Main config file for apache.
	- go to line 47 		\\Showing assign port number. 
	- Change the port to 82. and exit after save. 
systemctl restart httpd.service		\\To restart the service to implement the changes. 
	As a result the restart failed.
journalctl -xeu httpd.service		\\Analyze the logs for failure of httpd service.
semanage port -l | grep http		\\list the label for http
	- check the assigned port tcp.
semanage port -l | grep http port t	\\ To review the label for http port. 
semanage port -a -t http_port_t -p tcp 82		\\ The command to label the port change for http.
systemctl restart httpd.service			\\ After the change need to restart the service. 
systemctl status httpd		\\to check if the change port is reflecting. 
-- Check the firewall if the port is enabled or not. 
-- Relaod the services.
